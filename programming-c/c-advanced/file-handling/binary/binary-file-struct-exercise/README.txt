Exercise: Structures & Binary Files
-------------------------------------------------------------------------------

Es soll ein Array von "resistor_t" Instanzen in einem binary File gespeichert 
und gelesen werden.

A) Write binary File: write-file.c
    Öffnen Sie das binäre File "data.bin" und schreiben die das Array "data[]"
    in das File.
    Vergessen Sie nicht das File zu schließen. 

B) Read binary File: read-file.c 
    Öffnen Sie das binäre File "data.bin" und lesen die Daten in das "data[]"
    array. 
    Geben Sie die daten auf der Konsole aus:
        270 Ohms,  5 %
        120 Ohms,  2 %
        470 Ohms,  2 %
        1000 Ohms, 1 %
        680 Ohms,  5 %    
    Vergessen Sie nicht das File zu schließen.
