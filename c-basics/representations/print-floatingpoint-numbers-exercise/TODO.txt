Exercise: Print Floating-Point Numbers
-------------------------------------------------------------------------------

Schreiben Sie ein C Programm, das die gegebenen Variableninhalte in Form einer
Tabelle ausgibt:

        +---------------+-------------+
        | float         |       0.10  |
        | double        |     3.1415  |
        | double        | 1.2000e+03  |
        +---------------+-------------+

Tipp: Verwenden Sie die printf() Funktion.

Tipp: Siehe teiniker-lectures-computerscience/c-std-lib/stdio/printf
